# A1 Easy Pokémon Landing Page

A simple HTML and CSS landing page inspired by a Pokémon Gen I Pikachu design.

## Files

- `index.html` — main page structure
- `styles.css` — page styles and layout
- `pokemon-logo.png` — Pokémon logo image
- `pikachu.png` — Pikachu illustration
- `pokeball.png` — decorative Pokéball image

## Usage

1. Open `index.html` in your browser.
2. Make changes to `styles.css` to adjust the layout, colors, and spacing.

## Notes

- The design uses a split-screen layout with a yellow left panel and a cream right panel.
- The Pikachu artwork is positioned to overlap the two sections.
- The outer container includes rounded corners and a soft shadow.
